Definition 2.5.1.

A finite affine plane of order \(n\) (\(n\geq 2\)) is a model that satisfies the Axioms Axiom 2.2.5, Axiom 2.2.6, Axiom 2.2.7, Axiom 2.2.21 and the following additional Axioms:
Axiom 2.5.2.
Given a line \(L\) and a point \(A\text{,}\) there is a unique line containing \(A\) passing through \(L\text{.}\)
Axiom 2.5.3.
There is at least one line \(L\) containing exactly \(n\) points.
