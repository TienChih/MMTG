Definition 2.7.7.

The following is a list of some selected algebraic properties on a magma \((S,\cdot)\text{.}\)
\begin{align} \text{For all }x\in S: \amp\amp x\cdot x \amp= x \amp \text{Idempotent property}\tag{2.7.1}\\ \text{For all }x,y\in S: \amp\amp x\cdot y \amp = y\cdot x \amp \text{Commutative property}\tag{2.7.2}\\ \text{For all }x,y\in S: \amp\amp (x\cdot y)\cdot x \amp = x\cdot (y\cdot x) \amp \text{Flexible property}\tag{2.7.3}\\ \text{For all }x,y\in S: \amp\amp x\cdot x \amp = y\cdot y \amp \text{Unipotent property}\tag{2.7.4}\\ \text{For all }x,y,z\in S: \amp\amp (x\cdot y)\cdot z \amp = x\cdot (y\cdot z) \amp \text{Associative property}\tag{2.7.5}\\ \text{For all }x,y,z\in S: \amp\amp x\cdot (y\cdot z) \amp = (x\cdot y) \cdot (x\cdot z) \amp \text{Left-distributive property}\tag{2.7.6}\\ \text{For all }x,y,z\in S: \amp\amp (x\cdot y)\cdot z \amp = (x\cdot z) \cdot (y\cdot z) \amp \text{Right-distributive property}\tag{2.7.7}\\ \text{For all }x,y,z\in S: \amp\amp x\cdot y = x\cdot z \amp \text{ implies } y=z \amp \text{Left-cancel property}\tag{2.7.8}\\ \text{For all }x,y,z\in S: \amp\amp y\cdot x = z\cdot x \amp \text{ implies } y=z \amp \text{Right-cancel property}\tag{2.7.9}\\ \text{There exists }e\in S\text{ such that for all }x\in S: \amp\amp e\cdot x = x\cdot e \amp= x \amp \text{Identity property}\tag{2.7.10}\\ \text{For all }x\in S\text{ there exists }y\in S: \amp\amp x\cdot y = y\cdot x\amp = e \amp \text{Invertibility property}\tag{2.7.11} \end{align}
