    Section 3.1 Introduction to groups

    Definition 3.1.1.

    A group is a set \(G\) along with a binary algebraic operation \(\cdot\) that satisfies the following properties:
    1. Closure: For any \(a,b\) in \(G\text{,}\) it follows that \(a\cdot b\) in \(G\text{.}\)
    2. Associativity: For any \(a,b, c\) in \(G\text{,}\) it follows that \((a\cdot b)\cdot c = a\cdot (b\cdot c)\text{.}\)
    3. Identity: There is an element (usually denoted) \(e\) in \(G\text{,}\) such that for any \(a\) in \(G\text{,}\) we have that \(e\cdot a=a\) and \(a\cdot e=a\text{.}\)
    4. Inverses: For any \(a\) in \(G\text{,}\) we there is a \(a^{-1}\) in \(G\) so that \(a\cdot a^{-1}=e\text{.}\)

    Definition 3.1.8.

    A group is said to be abelian if for any \(a,b\in G, a\cdot b = b\cdot a\text{.}\)

    Definition 3.1.10.

    Given a group \(G\) a subset \(H\subseteq G\) is a subgroup of \(G\) if \(H\) is also a group with the same operation.

    Definition 3.1.13.

    Let \(\varphi:G\to H\text{.}\) If \(\varphi\) is a bijection such that \(\varphi(a\cdot b)=\varphi(a)\cdot\varphi(b)\) we say that \(\varphi\) is a isomorphism and \(G, H\) are isomorphic.
